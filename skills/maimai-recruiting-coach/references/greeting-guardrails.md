# 打招呼草稿护栏规则

更新时间：2026-03-23

## 1. 生成范围

- 默认仅对 `结论=可沟通` 的候选人生成草稿。
- 其他结论不自动生成，避免误触达。

## 2. 约束来源

- 约束字段：筛选模板中的 `招呼语要求`
- 常见规则：
  - 保密招聘
  - 不得出现岗位名（如 目标岗位负责人 / 目标岗位负责人）
  - 不得出现职级词

## 3. 生成策略

- 先生成基础草稿（匹配点 + 邀约下一步）
- 再做禁词替换与二次清洗
- 失败降级：
  - 不阻断候选人写库
  - 把失败原因写入候选人的 `结论原因`

## 4. 可配置参数

- `--greeting-from-template`
- `--template-name`
- `--filter-base-url`
- `--greeting-only-for`（默认 `可沟通`）
- `--overwrite-greeting`（显式覆盖已有草稿）
