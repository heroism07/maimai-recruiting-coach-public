# 筛选模板版本规则

更新时间：2026-03-23

## 1. 命名规范

- `模版名称` 统一格式：`<基础名>@vNNN`
- 示例：`关键岗位模板0323@v005`
- `模版基础名`：`关键岗位模板0323`
- `模版版本`：`5`

## 2. 拉取规则

- 输入基础名：自动命中该基础名的最新版本
- 输入完整版本名（含`@vNNN`）：精确命中该版本

## 3. 回写规则

- `append-only` 默认开启：
  - 条件有变化：新增新版本
  - 条件无变化：`no_change_skip`
- 新版本号：同基础名下 `max(version) + 1`
- 历史记录不覆盖、不删除

## 4. 追溯字段

- `父记录ID`：记录上一版本 record_id
- `变更摘要`：记录本次变更原因（建议传 `--change-note`）

## 5. 强制覆盖（仅人工）

- `--update-existing true` 才允许覆盖更新现有记录
- 强制覆盖时仍建议保留版本字段，避免混乱
