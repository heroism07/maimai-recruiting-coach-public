# 当前筛选表字段与脉脉控件映射（去冗余后）

## 直接对应脉脉筛选
- `模版名称` -> 模板索引主键（飞书侧，推荐）
- `模板名称` -> 模板索引兼容字段（可选）
- `职位需求` -> 记录用户原始职位要求（例如“行业A+甲方核心岗位+泛科技互联网”）
- `筛选策略` -> 记录由画像拆解出的筛选策略（must/nice/reject）与详情评估三维规则（`industry_core_background`、`party_a_core_background`、`domain_relevance`）及结论阈值
- `场景名称` -> 历史兼容字段（建议逐步迁移到 `职位需求`）
- `模版基础名` -> 模板归组索引（版本聚合）
- `模版版本` -> 模板版本号（整数）
- `父记录ID` -> 上一版本 record_id（追溯）
- `变更摘要` -> 本次修改说明（追溯）
- `是否启用` -> 是否参与拉取执行（飞书侧）
- `关键词` -> 搜索关键词输入框
- `关键词逻辑(所有/任一)` -> 关键词逻辑切换
- `城市口径` -> 城市筛选中的 `期望` / `现居` 复选
- `城市地区_列表` -> 城市多值（建议 `;` 分隔）
- `学历_最低` -> 学历自定义最低值
- `学历_最高` -> 学历自定义最高值
- `工作年限_最低(年)` -> 工作年限自定义最低
- `工作年限_最高(年)` -> 工作年限自定义最高
- `工作年限_在校应届` -> 在校/应届特殊开关
- `就职公司_范围` -> `正任职` / `曾任职`
- `就职公司_列表` -> 公司多值（建议 `;` 分隔）
- `职位名称` -> 职位名称多选
- `行业方向` -> 行业方向多选
- `毕业学校` -> 毕业学校筛选
- `专业` -> 专业筛选
- `性别` -> 性别筛选
- `年龄_最低` -> 年龄最低
- `年龄_最高` -> 年龄最高
- `期望月薪_最低K` -> 月薪最低（K）
- `期望月薪_最高K` -> 月薪最高（K）
- `家乡` -> 家乡筛选
- `智能筛选_公开求职意向_开关` -> 公开求职意向复选开关
- `智能筛选_公开求职意向_状态` -> 公开求职意向状态多选
- `智能筛选_近期有动向_开关` -> 近期有动向复选开关
- `智能筛选_近期有动向_范围` -> 近期有动向时间范围
- `智能筛选_有附件简历_开关` -> 有附件简历复选开关
- `智能筛选_有过意向_开关` -> 有过意向复选开关
- `智能筛选_企业号互动_开关` -> 企业号互动复选开关
- `智能筛选_企业号互动_类型` -> 企业号互动类型多选
- `排序方式` -> 综合/活跃/匹配排序
- `招呼语要求` -> 招呼语生成约束（业务规则字段）
- `备注` -> 业务备注

## 运行状态与可追溯（非脉脉筛选控件）
- `筛选模式` -> 读取优先级（结构化/兼容）
- `筛选条件JSON` -> 原始快照备份 + 结构化评估规则（建议包含 `detail_evaluation_rule.dimensions/weights/thresholds`）
- `结果人数` -> 执行结果
- `结果摘要` -> 执行摘要
- `执行状态` -> running/success/failed
- `最近执行时间` -> 最近执行时间戳
- `异常原因` -> 失败原因

## 索引与版本规则（新增）
- 读取模板时，支持基础名与完整版本名两种模式：
  - 输入基础名：自动命中最新版本
  - 输入 `@vNNN`：精确命中指定版本
- 脉脉筛选写回飞书时，默认按“同基础名最新版本”比对差异：有变化就新增一条记录，不覆盖历史记录。
- 如确需覆盖原记录，需显式使用 `update-existing` 模式（默认关闭）。

## 系统字段
- `文本` -> 多维表格主字段（系统字段，不参与筛选）
