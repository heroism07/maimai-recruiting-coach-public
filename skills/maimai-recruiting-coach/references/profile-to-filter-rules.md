# 画像转筛选规则（规则+LLM混合）

更新时间：2026-03-23

## 1. 输入协议（Profile）

建议输入字段：
- `职位名`
- `must` / `nice` / `reject`
- `目标公司`
- `行业方向`
- `地点`
- `工作年限最小值` / `工作年限最大值`
- `年龄最小值` / `年龄最大值`
- `薪资最小K` / `薪资最大K`
- `保密要求` / `招呼语要求`

## 2. 规则引擎输出（第一层）

规则引擎负责输出稳定结构化骨架：
- `关键词`、`关键词逻辑`
- 城市、学历、年限、年龄、薪资区间
- `就职公司_范围` + `就职公司_列表`
- 智能筛选默认值（建议：近期有动向=启用，近3个月）
- `招呼语要求`（若未输入则补默认保密文案）

## 3. LLM增强输出（第二层）

LLM只增强以下字段：
- `关键词`
- `关键词逻辑(所有/任一)`
- `招呼语要求`

约束：
- 不改动硬边界字段（年龄/年限/薪资区间）。
- LLM不可用时，直接回退规则结果，不中断流程。

## 4. 产物

统一生成两份：
- `runtime scenario`：用于应用到脉脉筛选
- 飞书筛选模板记录：通过版本化回写新增记录
